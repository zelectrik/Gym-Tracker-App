# Gym Tracker Pro

Fullstack fitness tracking application built with the same stack as the original pharma project.

## Stack

### Backend
- Node.js
- TypeScript
- Express
- Prisma
- PostgreSQL
- JWT Authentication
- Zod Validation
- Vitest
- Docker test database

### Frontend
- React
- Vite
- TypeScript

## Features

- Authentication with JWT
- Multiple users
- Gym management
- Workout templates
- Workout sessions tracking
- Duo training sessions
- Progress tracking
- Training history
- Admin dashboard
- Integration tests

## Project Structure

```txt
backend/
frontend/
```

## Development

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
